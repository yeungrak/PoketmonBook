# 🧬 PoketmonBook

포켓몬의 정보를 확인할 수 있는 포켓몬 도감 앱입니다.  
포켓몬 목록을 불러오고, 각 포켓몬의 상세 정보를 확인할 수 있습니다.

## 📱 주요 기능

- 포켓몬 리스트 조회
- 각 포켓몬의 상세 정보 확인 (이미지, 무게, 키 등)

## 🛠 기술 스택

- **Swift**
- **UIKit**
- **RxSwift**
- **SnapKit**

## 🧱 아키텍처

- **MVVM 구조**를 기반으로 구성하여 View와 비즈니스 로직을 분리
- 비동기 처리와 데이터 바인딩에 **RxSwift** 사용
